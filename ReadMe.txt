******************************
Available Files & Description 
******************************

AirBnB data:  Download http://data.insideairbnb.com/united-states/ny/new-york-city/2019-07-08/data/listings.csv.gz

Airbnb_Data_Challenge.ipynb --- Ipython notebook containing the Code, Analysis, Insights, Conclusions, Recommendations and Future Scope.

Airbnb_Data_Challenge.html --- HTML version of the Airbnb_Data_Challenge.ipynb This provides a better and an easy way to open and review the contents of python notebook.

zillow_air_DF.data --- pickled object file created to generate interactive dashboard

CapitalOne_Challenge.py --- Python script to generate an interactive dashboard.

*****************************************
Running CapitalOne_Challenge.py Dashboard 
*****************************************

1) All the unzipped files should be present in the same directory.
2) Install all the necessary packages.
3) Open anaconda prompt (for windows) or CLI (for mac).
4) Navigate to the unzipped folder containing all the files.
5) Run jupyter notebook to Launch Jupyter. Navigate to the appropriate folder in the Notebook and see the code.
6) From command line, run CapitalOne_Challenge.py 
	(command => python CapitalOne_Challenge.py)
7) Copy the http address generated from the above command. Paste it in to a browser to visualize the dashboard. (Example link => http://127.0.0.1:8000/)
